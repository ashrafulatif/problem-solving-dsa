# Problem Solving DSA (Java)

This repository contains simple Java solutions for common DSA beginner problems.

## Solved Problems

1. Reverse a String
2. Find the Largest Number in an Array
3. Check for Palindrome
4. Sum of Array Elements
5. Factorial Calculation
6. Even or Odd List (returns even numbers)
7. Fibonacci Sequence
8. Find the Minimum in an Array
9. Multiplication Table (up to 10)
10. Check Prime Number

All solutions are implemented in the `solver` class inside `solver.java`.

## How to Run

Use the following commands in the project folder:

```bash
javac solver.java
java solver
```
